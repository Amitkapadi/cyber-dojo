adding a new exercise

  • Create a new sub-directory under /var/www/cyberdojo/exercises/
    Example:
    $ cd /var/www/cyberdojo/exercises $ md FizzBuzz
  • Create a text file called instructions in this directory.
    Example:
    $ cd /var/www/cyberdojo/exercises/FizzBuzz $ touch instructions
  • Edit the instructions file to contain your instructions!
